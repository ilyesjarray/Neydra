CONNECT TO NLP ENGINE

Enter your LocalTunnel URL to connect to the Sentiment Decoder.

Run npx localtunnel --port 8000 in your terminal to generate a URL.

⚙ HOW TO USE

1 Install Node.js
Node.js is required to create the tunnel link. Download the LTS version for your system.
https://nodejs.org/en/download/
2 Download & Run Backend
Download the NEYDRA NLP Package below. Extract the .rar file, then double-click ultra_nlp_backend.exe to start the engine.

Instructions

  • Extract the .rar file to a folder
  • Double-click ultra_nlp_backend.exe to run
  • Wait for the console to show "ONLINE"
3 Create Public URL
Open a NEW terminal and run the tunnel command:
npx localtunnel --port 8000
4 Unlock Tunnel (Password)
Open the generated URL in your browser. It will ask for a password.
Password = Your Public IP.

Click below to find your IP, then paste it into the tunnel page.
🔍 FIND MY IP ADDRESS
5 Connect Dashboard
Once the tunnel page shows "status": "ONLINE", paste the URL into the connection panel and click Connect.
