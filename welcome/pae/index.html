CONNECT TO SERVER

Enter your LocalTunnel URL to connect to the Phoenix Engine.

Run npx localtunnel --port 8000 in your terminal to generate a URL.

HOW TO USE

  • 1
    Install Node.js

    Node.js is required to create the tunnel link. Download the LTS version for your system.

    ⬇ DOWNLOAD NODE.JS
  • 2
    Download PAE Package

    Download the bot files and extract them to a folder on your PC.

    ⬇ DOWNLOAD NEYDRA PAE PACKAGE
  • 3
    Run Backend Engine

    Open the extracted folder and double-click the application below to start:

    neydra_ultra.exe
  • 4
    Create Public URL

    Open a NEW Command Prompt and run the tunnel command. Copy the URL it gives you.

    npx localtunnel --port 8000
  • 5
    Unlock Tunnel (Password)

    Open the generated URL in your browser. It will ask for a password.

    Password = Your Public IP. Click the button below to find your IP, then paste it into the tunnel page.

    🔍 FIND MY IP ADDRESS
  • 6
    Connect Dashboard

    Once the tunnel page shows "status: ONLINE", paste the URL here and click Connect.
